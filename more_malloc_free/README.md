# more_malloc_free

This directory contains C programs and exercises related to dynamic memory allocation using `malloc`, `free`, and related functions.

## Usage

Compile the C files using `gcc` and run the resulting executables.

## Files

- `0-malloc_checked.c`
- `1-string_nconcat.c`
- `2-calloc.c`
- `3-array_range.c`
